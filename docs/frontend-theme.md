# Frontend Theme

DeskCrafter uses the discontinued CherryIDE landing project as theme inspiration, not as a direct copy.

## Visual Thesis

Dark Linux workstation utility with gunmetal surfaces, cherry red action states, restrained glow, sharp typography, and calm operational density.

## Reused Ideas

- Black and gunmetal base colors.
- Cherry red as the single primary accent.
- Metallic buttons and subtle focused borders.
- Strong full-bleed product website hero.
- Dense desktop app workspace with left navigation, central work area, and right inspector.

## Avoided Ideas

- Heavy CherryIDE-specific 3D IDE scenes inside the desktop app.
- Marketing hero layouts inside the product workspace.
- Dashboard mosaics and decorative panels that do not support a workflow.
